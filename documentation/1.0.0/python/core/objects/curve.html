class kraken.core.objects.curve.Curve(name, parent=None)¶

Bases: kraken.core.objects.object_3d.Object3D

Curve object.

__init__(name, parent=None)¶
getCurveData()¶

Returns the data of the curve.

Return: List, dictionaries defining each sub-curve of this curve.

setCurveData(data)¶

Sets the curve data.

Arguments: data – List, dictionaries defining each sub-curve of this curve.

Return: True if successful.

appendCurveData(data)¶

Appends sub-curve data to this curve.

Arguments: data – List, dictionaries defining each sub-curve being added to this curve.

Return: True if successful.

checkSubCurveIndex(index)¶

Checks the supplied index is valid.

Arguments: index – Integer, sub-curve index to check.

getNumSubCurves()¶

Returns the number of sub-curves on this object.

Return: Integer, number of sub-curves in this object.

getSubCurveClosed(index)¶

Returns whether the sub-curve is closed or not.

Arguments: index – Integer, index of the sub-curve to query.

Return: True or False if the sub-curve is closed.

getSubCurveData(index)¶

Get the sub-curve data by it’s index.

Arguments: index – Integer, index of the sub-curve to get the data for.

Return: Dict, data defining the sub-curve.

setSubCurveData(index, data)¶

Sets the sub-curve data.

Arguments: index – Integer, index of the sub-curve to get the data for. data – Dict, defining the sub-curve data.

Return: True if successful.

removeSubCurveByIndex(index)¶

Removes a sub-curve by its index.

Arguments: index – Integer, Index of the sub-curve to remove.

Return: True if successful.
