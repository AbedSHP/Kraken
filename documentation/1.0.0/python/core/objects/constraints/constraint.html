Returns the whether the constraint should maintain offset when it’s built or not.

Return: Boolean, whether the constraint should maintain offset or not.

Sets the constraint to maintain offset when creating the constraint.

Arguments: value – Boolean, whether the constraint should maintain offset or not.

Return: True if successful.

Return: True if successful.

Return: True if successful.

Arguments: kObject3D – kObject3D, Kraken 3D object. index – Integer, index of the constraint to set.

Return: True if successful.

removeConstrainerByIndex(index)¶

Return: True if successful.
